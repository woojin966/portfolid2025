퍼블리싱 : 선임 퍼블리셔를 서포트하며 마크업, 네이밍 규칙 정립 및 시멘틱 구조 설계를 담당하였습니다.

사용스킬

HTML,CSS

1프로젝트 개요

선임 퍼블리셔가 주도한 프로젝트에 서포트 역할로 참여하여, 마크업 구조와 클래스 네이밍 규칙을 정립하고, 코드 효율성을 높이기 위한 퍼블리싱 작업을 진행하였습니다. 프로젝트 전반에 걸쳐 시멘틱 마크업 및 컴포넌트 기반 퍼블리싱 방식을 적용하였습니다.

2주요 구현 내용

Flexbox와 Grid 레이아웃을 적절히 조합하여 화면 구성의 일관성과 유연성을 확보하였으며, 마크업과 스타일 코드 전반에 걸쳐 통일성을 유지함으로써 효율적인 퍼블리싱 작업이 이루어질 수 있도록 구조 설계에 심혈을 기울였습니다.

3작업 중 어려움 & 해결법

문제 초기에는 선임 퍼블리셔의 코드 작성 방식과 마크업 구조에 익숙하지 않아 협업 과정에서 혼란이 있었고, 특히 Grid 레이아웃에 대한 이해도가 낮아 레이아웃 구성 시 시행착오가 반복되어 작업 속도와 완성도에 영향을 주었습니다.

해결 지속적인 소통을 통해 코드 규칙과 구조를 명확히 이해하고, 마크업과 클래스 네이밍에 일관성을 부여하여 협업 효율을 높였습니다. 반복되는 UI 요소는 컴포넌트로 분리하고, 시멘틱 태그를 적극 활용하여 코드 중복을 줄이며 유지보수성을 향상시켰습니다. 특히 Grid 레이아웃을 다양한 상황에 직접 적용해보며 활용법을 익히고, 실무에 적용 가능한 수준으로 퍼블리싱 역량을 성장시킬 수 있었습니다.

4결과 및 성과

코드 구조에 일관성을 부여하여 협업 시 가독성과 효율성 향상

컴포넌트화 및 시멘틱 마크업을 통해 유지보수성과 확장성 개선

선임 퍼블리셔의 코드 스타일을 이해하고 흡수하며 실무 퍼블리싱 역량 향상
