📌 Scroll Interaction UI – 스크롤 반응형 고정 UI 구현

기간: 2024.02 (약 2주)
참여도: 퍼블리싱 100%
역할: 고정형 UI 설계 및 마크업, 인터랙션 구현, 성능 최적화

1. 프로젝트 개요

사용자의 스크롤 위치에 따라 UI 요소가 반응하도록 구현한 마이크로 인터랙션 프로젝트입니다. 고정된 내비게이션과 아이콘 상태 변화 등 시각적 피드백을 통해 사용자 경험을 강화하는 데 중점을 두었습니다.

2. 사용 기술 / 개발환경

  • HTML5 SCSS jQuery JavaScript
  • position: sticky를 활용한 고정 UI
  • 스크롤 이벤트 기반 동적 클래스 토글

3. 주요 구현 내용

  • 스크롤 시 섹션에 진입하면 해당 아이콘에 `.active` 클래스 자동 적용
  • CSS 및 jQuery로 상태 변화에 따른 애니메이션 구현
  • 기기의 해상도에 따른 조건부 마크업 처리 (`.mo_hidden` 클래스 등)

▲ 스크롤 위치에 따라 아이콘이 활성화되는 고정형 UI

4. 작업 중 어려움 & 해결법

  • 문제: `position: sticky`가 특정 브라우저에서 작동하지 않음
    해결: 부모 요소의 `overflow`, 높이 설정 등을 점검하여 레이아웃 구조 수정
  • 문제: 스크롤 이벤트 다중 호출로 인한 성능 저하
    해결: `throttle` 기법 도입으로 이벤트 최적화

5. 결과 및 성과

  • 스크롤 인터랙션 적용으로 콘텐츠 몰입도 및 UI 이해도 향상
  • 퍼블리셔 관점에서 동적 마크업 구조 경험 축적
  • 디자인 의도를 살리면서도 가벼운 UI 구현 성공
