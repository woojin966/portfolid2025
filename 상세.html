📱 Project Name – 모바일 앱 리디자인

기간: 2024.03 – 2024.04 (약 4주)
참여도: 퍼블리싱 100%
역할: 마크업 구조 설계, 반응형 구현, 접근성 개선, 인터랙션 개발

1. 프로젝트 개요

기존 앱의 사용성 문제를 해결하고, 시각적 일관성을 높이기 위한 리디자인 프로젝트입니다. 사용자 피드백을 기반으로 UX 흐름을 개선했고, 디자인 가이드를 기반으로 반응형 웹앱 형태로 퍼블리싱을 진행했습니다.

2. 사용 기술 / 개발환경

  • HTML5 SCSS Vanilla JS GSAP Zeplin
  • 모듈화된 SCSS 구조로 컴포넌트 재사용성 확보
  • Zeplin을 통한 디자이너와 협업

3. 주요 구현 내용

  • 메인 – 히어로 섹션 애니메이션 (GSAP)
  • 스크롤 트리거 기반 콘텐츠 등장 애니메이션 (IntersectionObserver)
  • 전 페이지 반응형 구현 (모바일/태블릿 대응)
  • 시멘틱 태그 + aria 속성으로 접근성 향상

▲ 메인 페이지 히어로 섹션 – GSAP 애니메이션 적용

4. 작업 중 어려움 & 해결법

  • 문제: 모바일 환경에서 배경 이미지 위치 깨짐 발생
    해결: `object-fit` 과 media query 조정으로 디바이스별 이미지 위치 조정
  • 문제: IntersectionObserver에서 여러 개 섹션 중복 실행
    해결: `once: true` 속성과 `unobserve()` 활용으로 처리

5. 결과 및 성과

  • 크로스브라우징 100% 완료 (Chrome, Safari, Edge)
  • 누락 기능 및 디자인 피드백 전부 일정 내 처리
  • 접근성 점수 Lighthouse 기준 95점 이상 확보
